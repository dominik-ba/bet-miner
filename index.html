Bet Miner - FIFA WM 2018

This is a REST server to provide data access to mined betting data.

Currently only group stage data is available.

Data comes from:

  • bet-at-home
  • bwin
  • interwetten

Some Endpoints include:

  • GET /bets?before=timestamp?limit=1000 Retrieve 1000 Bets max, sorted descending by date
  • GET /bets/:id Retrieve a specific Bet
  • GET /games Retrive all games where data was mined for
  • GET /games/:id Retrieve a specific game
